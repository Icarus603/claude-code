## 概述

Buddy 是 Claude Code 內置的虛擬寵物系統。在 REPL 中通過 `/buddy` 命令可以孵化一隻隨機生成的寵物伴侶，它會出現在輸入框旁邊，陪伴你的編碼過程。

> Feature Flag: `FEATURE_BUDDY=1`

## 啓用方式

```bash
FEATURE_BUDDY=1 bun run dev
```

孵化窗口：2026 年 4 月 1-7 日期間啓動時，會在 REPL 頂部顯示彩虹色的 `/buddy` 提示。4 月 7 日之後命令仍然可用，但不再自動提示。

## 命令

| 命令 | 說明 |
|---|---|
| `/buddy` | 查看當前寵物信息和屬性 |
| `/buddy hatch` | 孵化一隻新寵物（首次使用） |
| `/buddy rehatch` | 重新隨機生成寵物（替換現有） |
| `/buddy pet` | 擼寵物，觸發愛心動畫 |
| `/buddy mute` | 靜音寵物（隱藏） |
| `/buddy unmute` | 取消靜音 |

孵化時基於種子隨機決定，存在極低概率出現 Shiny（閃光）變體。

### 屬性值

每隻寵物擁有 5 項屬性（0-100）：

- **DEBUGGING** — 調試能力
- **PATIENCE** — 耐心程度
- **CHAOS** — 混亂指數
- **WISDOM** — 智慧值
- **SNARK** — 毒舌度

### 外觀

每隻寵物還有隨機的外觀配件：

- **眼睛**: `·` `✦` `×` `◉` `@` `°`
- **帽子**: none, crown, tophat, propeller, halo, wizard, beanie, tinyduck

## 資料存儲

寵物信息存儲在 `~/.claude.json` 的 `companion` 字段中。寵物的外觀屬性（物種、稀有度、屬性值等）基於用戶 ID 的哈希確定性生成，不可通過編輯設定文件來篡改稀有度。
